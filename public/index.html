Provide The Operation

You have to much this pattern : (add|concat|sortasc|sortdesc) XX YY ZZ YYY

The numbers must be separated by spaces and could be 1, 2 or more digits

open/closed principle: To be able to extend the operations:

  1. Add the operand to the regex rule 
    const regex = /^((add|concat|sortasc|sortdesc)\s+\d+(?:\s+\d+){0,5}|(add|concat|sortasc)\s+\d+(?:\s+\d+){0,5}\s)$/;
  2. Adjust the help text
  3. Implement the method in the RequestHandler class that auto detect if the it implement the operation or not (handler())
